Detailed Description

Obstacle avoidance library.

GENERAL: Library is intended for robots utilizing range finder mechanisms, such as laser or ir range finders.

USAGE NOTES: When frames are added utilizing addFrame, the solutionUpdate function must be called, in respective sequence, so that outputs can be generated with respect to expected input frame count.
